## <span class="c7">Project Overview</span>

# <span class="c6">Project Management Plan</span>

## <span class="c7">Tasks</span>

*   <span class="c2">Develop GUI</span>
*   <span class="c2">Develop source code for discord bot</span>
*   <span class="c2">Create a command list for users to input</span>

## <span class="c7">Description</span>

*   <span class="c2">An automatic, modular Discord bot for the Discord GUI that can assist users in a variety of tasks, most notably storing table information.</span>

## <span class="c7">Deliverables and Milestones</span>

*   <span class="c2">SPMP v0.1a</span>

*   <span class="c2">Internet access, computer, Discord software, and accounts.</span>

## <span class="c7">Dependencies and Constraints</span>

*   <span class="c2">We have a limited amount of time and resources that we can put into this project. Certain features of the bot may be left out depending on our development as the project progresses.</span>

# <span class="c6">Additional Information</span>

## <span class="c7">Background Information</span>

*   <span class="c34 c35">The social media bot is a program within social media that functions as an automated user on the server. Depending on the systems present in a social media application or website, a properly programmed bot can autonomously perform multiple tasks on social media at greater speed than a human user. Bots also present the ability to interact with people via messaging or posting and even respond to basic programmed commands sent to it via audio or text.</span>
